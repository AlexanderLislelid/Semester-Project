# Community Science Museum

A modern, responsive website designed for the Community Science Museum.  
The site aims to engage children and young teens (ages 7–15) by presenting science through interactive, accessible and visually appealing content.

## Features

- Fully responsive layout for mobile, tablet and desktop  
- Modern design inspired by the museum’s Figma prototype  
- Dedicated sections for exhibitions, events and visitor information   
- Optimized images
